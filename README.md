dotfiles
========
my dotfiles

# how to install

## install
execute command below
```
# git clone https://github.com/kegamin/dotfiles.git dotfiles
# cd dotfiles
# ./setup_env.sh
```
### .gitconfig
should modify name and email.

## backup
backup dotfiles automatically to $HOME/dotbackup/${dotfile}.yyyymmddhhmmss
